# Card Asset Guide

This project is set up for composable troop card art.

## Recommended production format

- Runtime format: PNG
- Size: 384x480 px per card (4:5)

## Current placeholders

This folder set includes starter SVG placeholders and matching PNG exports:

- `assets/cards/troops/components/troop_frame.svg`
- `assets/cards/troops/components/value_badge.svg`
- `assets/cards/troops/components/symbol_<color>.svg`
- `assets/cards/troops/backs/troop_back.svg`
- `assets/cards/tactics/backs/tactic_back.svg`
- `assets/cards/flags/flag_card.svg`

The runtime currently loads the PNG versions of those same files.

## Troop symbols

- red
- orange
- yellow
- green
- blue
- purple

## Troop values

- 1 through 10 for each symbol

Total troop cards: 60.
